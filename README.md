#  Sistema de Gestión de Heladería

##  Descripción del sistema

#Este proyecto consiste en el desarrollo de una aplicación de consola en C# que permite gestionar las operaciones básicas de una heladería.

#El sistema permite administrar:
#- Clientes
#- Helados
#- Pedidos

#El objetivo es aplicar los conceptos de Programación Orientada a Objetos (POO) y el uso de persistencia de datos mediante archivos CSV.


## 🧠 Conceptos de POO implementados

#- Herencia: Cliente hereda de Persona  
#- Asociación: Pedido está relacionado con Cliente  
#- Agregación: Helado tiene un Sabor  
#- Composición: Pedido contiene Detalles  
#- Interfaz: ICrud para operaciones CRUD  


##  Funcionalidades

#El sistema permite:
#- Crear registros
#- Consultar registros
#- Actualizar registros
#- Eliminar registros


## Persistencia

#Se utilizarán archivos CSV con la librería CsvHelper.
